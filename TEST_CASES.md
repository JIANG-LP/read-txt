## 自动化测试脚本

### 手动测试流程

```
1. 启动 VSCode 扩展开发宿主
2. 按 Ctrl+Shift+T 激活插件
3. 选择 example.txt 文件
4. 验证状态栏显示第一行
5. 点击"下一行"按钮 5 次
6. 点击"上一行"按钮 3 次
7. 使用快捷键 Ctrl+Shift+T 隐藏
8. 再次按 Ctrl+Shift+T 显示
9. 修改 example.txt 添加新行
10. 验证自动更新
11. 选择其他 txt 文件
12. 验证切换成功
13. 重启 VSCode
14. 验证自动加载上次文件
```

### 性能测试脚本

```bash
# 测试大文件加载
wc -l large_file.txt  # 查看行数
time code --extensionDevelopmentPath=.  # 测试加载时间

# 监控内存
ps aux | grep 'Electron'
```
