# 功能演示说明 📸

## 视觉效果描述

由于无法直接展示 GIF 或截图，以下是详细的功能演示文字描述，帮助你想象插件的实际效果。

## 场景一：首次激活插件

### 操作步骤
1. 打开 VSCode
2. 按下 `Ctrl+Shift+T`

### 预期效果
```
[状态栏] ──────────────────────────────────────────────

弹出文件选择对话框:
┌─────────────────────────────────┐
│   选择 TXT 文件                    │
├─────────────────────────────────┤
│ 📁 Desktop                      │
│ 📁 Documents                    │
│ 📄 example.txt                  │
│ 📄 notes.txt                    │
├─────────────────────────────────┤
│          [打开]  [取消]          │
└─────────────────────────────────┘
```

## 场景二：加载文件后的状态栏

### 选择 example.txt 后

```
[状态栏] ──────────────────────────────────────────────

← 上一行    📄 这是第一行内容...    下一行 →
                            ↑
                        当前显示区域
```

**详细说明：**
- **左侧按钮**: `$(chevron-left) 上一行` - 点击显示上一行
- **中间文本**: `$(file-text) 这是第一行内容...` - 当前内容
- **右侧按钮**: `下一行 $(chevron-right)` - 点击显示下一行

### 悬停提示（Tooltip）

当鼠标悬停在中间文本区域时：

```
┌──────────────────────────────────────────┐
│ 📄 文件：example.txt                     │
│ 📊 进度：第 1 / 10 行                    │
│ 📍 路径：d:\project\read-demo\example.txt│
│                                          │
│ 点击切换文件 | ← 上一行 | → 下一行       │
└──────────────────────────────────────────┘
```

## 界面元素详解

### 状态栏布局

```
VSCode 窗口底部
┌─────────────────────────────────────────────────────┐
│ ← 上一行  │  📄 内容显示  │ 下一行 → │  Ln 1, Col 1 │
│    ↑      │      ↑       │    ↑     │      ↑       │
│  左按钮   │  文本区域    │ 右按钮   │ VSCode 原生  │
└─────────────────────────────────────────────────────┘
```

### 图标说明

- `$(file-text)` 📄 - 文本文件图标
- `$(chevron-left)` ← - 左箭头图标
- `$(chevron-right)` → - 右箭头图标

### 颜色主题

默认适配 VSCode 当前主题：
- **浅色主题**: 深色文字，浅色背景
- **深色主题**: 浅色文字，深色背景
- **高对比度主题**: 自动调整对比度
