## 项目概览

已成功从 0 开始搭建了一个功能完整的 VSCode 插件项目：**TXT Status Bar Reader**

### 📦 项目名称
- **Name**: txt-status-bar-reader
- **DisplayName**: TXT Status Bar Reader
- **Version**: 1.0.0

### 🎯 核心功能

在 VSCode 状态栏中显示指定 TXT 文件的内容，每次仅显示一行文本，支持通过点击状态栏中的左右方向按钮或快捷键来切换显示上一行和下一行文本内容。

## 🎨 技术架构

### 设计模式
- **单一职责原则**: 每个类只负责一个功能模块
- **观察者模式**: 文件系统监听器
- **生命周期管理**: 资源的正确创建和销毁

### 核心类说明

#### 1. TxtFileReader
```typescript
职责：文件读取和管理
功能:
  - loadFile(filePath)     // 加载文件
  - getCurrentLine()       // 获取当前行
  - getPreviousLine()      // 获取上一行
  - getNextLine()          // 获取下一行
  - dispose()              // 清理资源
特性:
  - 文件变化监听
  - 自动刷新支持
  - 循环浏览
```

#### 2. StatusBarManager
```typescript
职责：状态栏 UI 管理
功能:
  - show()                 // 显示状态栏
  - hide()                 // 隐藏状态栏
  - loadFile(filePath)     // 加载并显示文件
  - previousLine()         // 切换到上一行
  - nextLine()             // 切换到下一行
组成:
  - previousButton         // 上一行按钮
  - statusBarItem          // 文本显示项
  - nextButton             // 下一行按钮
```

#### 3. Extension (主入口)
```typescript
职责：初始化和协调
功能:
  - activate()             // 插件激活
  - registerCommands()     // 注册命令
  - deactivate()           // 插件停用
```

## 🚀 使用流程

### 首次使用
1. 按 `Ctrl+Shift+T` 激活插件
2. 选择要读取的 TXT 文件
3. 状态栏显示第一行内容

### 日常操作
- **切换行**: 点击左右按钮或使用快捷键
- **隐藏/显示**: 按 `Ctrl+Shift+T`
- **换文件**: 点击状态栏文本区域

### 配置选项
```json
{
  "txtReader.filePath": "",              // 文件路径
  "txtReader.refreshInterval": 0         // 刷新间隔（毫秒）
}
```
