# 自动翻页功能使用说明

## 功能概述

自动翻页功能允许 TXT 文件内容按照设定的时间间隔自动切换到下一行，适合需要持续滚动查看内容的场景。

## 使用方法

### 方法一：通过命令面板启用

1. 按 `Ctrl+Shift+P` (Mac: `Cmd+Shift+P`) 打开命令面板
2. 输入 `TXT Reader: 切换自动翻页` 并回车
3. 如果是首次使用，选择一个翻页间隔（1 秒、2 秒、3 秒、5 秒、10 秒）
4. 如果已经启用，会直接关闭自动翻页

### 方法二：通过配置文件设置

在 VSCode 的设置中配置 `txtReader.autoPageInterval`：

```json
{
  "txtReader.autoPageInterval": 3000
}
```

- 设置为 `0` 表示禁用自动翻页
- 设置为 `1000` 表示每秒翻一页
- 设置为 `5000` 表示每 5 秒翻一页

## 工作逻辑

1. **启动条件**：只有当状态栏显示时，自动翻页才会生效
2. **手动干预**：手动切换行（上一行/下一行）后，定时器会重置，避免立即触发自动翻页
3. **隐藏暂停**：隐藏状态栏时，自动翻页会自动停止
4. **配置持久化**：设置的间隔会保存在 VSCode 全局配置中，下次启动仍然有效

## 快捷键建议

你可以为自动翻页功能添加快捷键，在 `keybindings.json` 中添加：

```json
{
  "key": "ctrl+shift+a",
  "mac": "cmd+shift+a",
  "command": "txtReader.toggleAutoPage",
  "when": "editorTextFocus"
}
```

## 注意事项

- 自动翻页只在状态栏可见时运行
- 文件总行数为 0 或未加载文件时，自动翻页不会启动
- 可以通过设置 `autoPageInterval` 为 `0` 来完全禁用此功能
